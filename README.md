# AI-Safe Public Project Context

This repository contains the AI-safe public project context file for Import Translation Manager.

`public-context.md` is auto-generated from a reviewed file in the private source repository. It is intended for custom GPT and Codex prompt-generation workflows that need live, high-level project context without private implementation detail.

The same reviewed context is also available through the generated JSON endpoint at `api/public-context.json`. Custom GPT actions can import the generated OpenAPI document at `openapi.json`.

No sensitive implementation details, environment configuration, secrets, private infrastructure notes, customer data, or internal documentation are intentionally included here. If sensitive content is found, treat it as an incident: remove it from this public repository, rotate any affected credentials, and fix the private source file before republishing.
